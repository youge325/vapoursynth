应用程序和库
==========================

应用程序
############

* `D2V Witch <https://github.com/dubhater/D2VWitch>`_ -- 创建临时打开 d2vsource 的索引
* `Hybrid <https://www.selur.de/>`_ `(Doom9) <https://forum.doom9.org/showthread.php?t=153035>`__ -- 支持 VapourSynth 的编码图形界面
* `mpv.net <https://github.com/stax76/mpv.net>`_ -- 内置 VapourSynth 的媒体播放器
* `SmoothVideo Project <https://www.svp-team.com/wiki/Main_Page>`_ -- 用于实时帧插值的插件/视频播放器组件
* `StaxRip <https://github.com/staxrip/staxrip>`_ -- 提供增强的 VapourSynth 脚本支持的编码图形界面
* `VapourSynth Editor <https://github.com/YomikoR/VapourSynth-Editor>`_ `(Doom9) <https://forum.doom9.org/showthread.php?t=170965>`__ -- 支持语法补全和快速预览的编辑器
* `VapourSynth Editor 2 <https://bitbucket.org/gundamftw/vapoursynth-editor-2/>`_ `(Doom9) <https://forum.doom9.org/showthread.php?t=181708>`__ -- VapourSynth 编辑器的精神续作
* `VirtualDub2 <https://sourceforge.net/projects/vdfiltermod/>`_ `(Doom9) <https://forum.doom9.org/showthread.php?t=172021>`__ -- 增加了VirtualDub的高位深度色彩空间支持，可用于预览
* `vspreview <https://github.com/Irrational-Encoding-Wizardry/vs-preview>`_ -- 使用Python编写的高级独立预览器，功能丰富
* `vspreview-rs <https://github.com/quietvoid/vspreview-rs>`_ `(Doom9) <https://forum.doom9.org/showthread.php?t=183903>`__ -- 极简 VapourSynth 脚本预览器
* `Wobbly <https://github.com/Jaded-Encoding-Thaumaturgy/Wobbly>`_ -- IVTC 助手，类似于 YATTA

库
#########

* `VapourSynth.nim <https://forum.doom9.org/showthread.php?p=1905351>`_ -- Nim 编程语言绑定
* `vapoursynth-rs <https://crates.io/crates/vapoursynth>`_ -- VapourSynth Rust 封装
* `vsxx <https://github.com/sekrit-twc/vsxx>`_ -- VapourSynth C++ API 封装
