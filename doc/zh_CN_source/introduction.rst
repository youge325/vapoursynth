简介
============

VapourSynth 是一个视频处理应用程序，也可以是一个插件，或者一个库。由于它是用 C++ 编写的核心库，并创建用于视频脚本的 Python 模块，所以很难精确定义它。它诞生于我开始思考 Avisynth 替代设计方案之时，大部分代码在 3 个月内完成。

本软件基于 `Avisynth <http://www.avisynth.nl>`_ 的启发，旨在成为面向 21 世纪的重写版本，充分利用自 90 年代末以来计算机技术的进步。与 Avisynth 相比，主要特性包括：

   * 多线程 - 具有良好扩展性的帧级多线程
   * 通用色彩空间 - 可在运行时指定新的色彩空间
   * 逐帧属性 - 可为帧附加额外的元数据
   * 基于Python - 脚本部分作为Python模块实现，需要学习专门的语言
   * 支持格式变化的视频 - 有些视频无法保持单一格式或帧尺寸。VapourSynth 能够处理任何类型的变化
   * 兼容大量现有的 Avisynth 插件
